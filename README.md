# Ada Directory

**Ada Directory** adalah JOSM Tagging Preset untuk pemetaan OpenStreetMap di Indonesia, dirancang untuk mempercepat tagging sekaligus menjaga konsistensi data dengan preset
yang luas, templat lokal, dan pengisian alamat otomatis.

> [!WARNING]
> Masih dalam tahap pengembangan. Cakupan saat ini difokuskan pada wilayah **Kota Cirebon**.
> 
> Selalu periksa hasil edit sebelum upload ke OpenStreetMap

## Fitur

### Alamat Otomatis
Pilih kelurahan — kode pos, kecamatan, kota, provinsi, dan negara terisi otomatis.

### Preset Lengkap
Mencakup banyak preset bisnis, fasilitas, dan infrastruktur yang umum ditemui di Indonesia.

### Templat Lokal
Setiap preset dilengkapi field yang telah disesuaikan untuk Indonesia.

## Instalasi

1. Buka JOSM, masuk ke **Edit → Preferences → Tagging Presets**
2. Klik **+** dan masukkan URL berikut:

```
https://raw.githubusercontent.com/ivandhfr/Ada-Directory/refs/heads/main/Ada-Directory.xml
```

3. Klik **OK**
